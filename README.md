Java Development Internship at Cognifyz Technologies.

through practical, hands-on tasks divided into multiple levels.

- Organization: Cognifyz Technologies
- Domain: Java Development
- Levels Completed: Level 1, Level 2, Level 3

## Key Concepts Covered
- Core Java
- Object-Oriented Programming (OOP)
- String Manipulation
- File Handling
- Multithreading
- Basic Networking
- User Input and Output
- Problem Solving

## Project Structure
- Each level is organized in a separate folder
- Each task is implemented in an individual Java file
- Code is written with clarity and proper comments

## Purpose
This repository serves as proof of work and learning achieved
during the internship and demonstrates practical Java development skills.
